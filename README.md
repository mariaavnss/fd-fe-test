# Portfolio of Maria Vanessa Salim
The hosted app here: https://fd-landingpage-vane.netlify.app/

## Set up
```
# Install Dependencies
$ yarn install

# Serve with hot reload at localhost:3000
$ yarn run start

# Build for production and launch server
$ yarn build
```
